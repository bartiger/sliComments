# sliComments #

A powerful and slick comments component for Joomla!

## Requirements ##

* Joomla! 1.6 or 1.7

## Features ##

* Ajax
* Akismet plugin
* Approval system
* Ratings
* BBCode and/or HTML
* Emoticons
* ACL
* Integration with Kunena and JomSocial (profile)
* Gravatar
* No powered by or hidden links
* Easy updates through the Extension Manager
